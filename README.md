# babynamegen
CLI to help you find a baby name. Uses 2022 SSN names data.

# usage:

- git clone repo. 
- run `go run main.go`

```sh
$ go run main.go 
babynamegen helps you find a baby name. uses 2022 SSN names data.
enter gender (boy/girl): boy
pick from the top how many names? (default 1000): 50
press enter (or type 'girl')

Random boy name: Luca (rank: 28)
press enter (or type 'girl')

Random boy name: Daniel (rank: 14)
press enter (or type 'girl')

Random boy name: Oliver (rank: 3)
press enter (or type 'girl')

Random boy name: Grayson (rank: 37)
press enter (or type 'girl')
```
